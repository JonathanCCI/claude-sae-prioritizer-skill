# Scoring Criteria

## Formula

```
score = (need × 0.30) + (solution × 0.25) + (timing × 0.20) + (news × 0.15) + (contacts × 0.10)
```

Focus accounts: +0.5 bonus (max 10)

## Criteria

### CI/CD Need (30%)
- 9-10: 100+ engineers, microservices, complex deployments
- 7-8: Multiple dev teams, significant automation needs
- 5-6: Moderate development activity
- 3-4: Limited software development
- 1-2: Minimal technical footprint

### Current Solution (25%)
- 9-10: Jenkins (migration target), legacy CI, known pain
- 7-8: Mixed tooling, consolidation opportunity
- 5-6: GitHub Actions at scale (pricing pressure)
- 3-4: Established CI, no clear dissatisfaction
- 1-2: Already CircleCI or locked-in competitor

### Market Timing (20%)
- 9-10: Active Jenkins migration, GitHub Actions cost concerns
- 7-8: Cloud migration, DevOps transformation
- 5-6: General modernization initiatives
- 3-4: No urgency identified
- 1-2: Recently evaluated CI/CD

### Company News (15%)
- 9-10: New funding, new CTO/VP Eng, rapid growth
- 7-8: Product launches, engineering hiring
- 5-6: Steady growth
- 3-4: No notable news
- 1-2: Layoffs, budget cuts

### Reachable Contacts (10%)
- 9-10: Multiple contacts, warm intro possible
- 7-8: Key personas identified
- 5-6: Some contacts findable
- 3-4: Limited presence
- 1-2: No contacts found

## Target Personas

VP Engineering, Platform Lead, DevOps Manager, Engineering Manager, Developer Experience Lead
